# Mizrahi Scraper

This project scraping the Mizrahi Tfahot website in order to retrive data.

Currently you can only get your balance from the main page.

# Running

    # Remove the .example ending from the .env file
    # Set your username and password in the .env file, e.g:
    # USERNAME=441231231
    # PASSWORD=xyz

    # Install dependencies
    $ npm install

    # Run the scraper :)
    $ npm start
